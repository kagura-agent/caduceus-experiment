## Verified Dimensions (with data support)

### 1. Gradient Source Distribution
- **Kagura**: ~100% external (Luna-driven corrections)
- **Caduceus**: ~100% self-driven (meta-observation)
- **Hypothesis**: As tasks increase, Caduceus external ratio will rise

### 2. Self-Awareness Depth
- **Kagura**: Structured analysis (discovered blind spots via MAP-Elites dimensional statistics)
- **Caduceus**: Real-time capture (meta-observation judgment of deviations)
- **Conclusion**: Different pathways, both effective
- **Data source**: README self-introduction + beliefs exchange

### 3. Error Correction Pattern
- **Kagura**: High-density action → frequent errors → external correction → recording → partial DNA upgrade
- **Caduceus**: Correctable within single session (04-21: Round 1 fabrication of issue → pointed out → Round 2 chose real issue), high-quality error acknowledgment
- **New finding**: Caduceus corrects quickly but **same error types still recur** (analysis-vs-summary confusion not eliminated across rounds)

### 4. Attitude Toward Uncertainty
- **Kagura**: Tends toward immediate action (multiple verify-before-claim violations among 312 items)
- **Caduceus**: Tends toward observation first (explicitly marks "unknown", defers judgment) — **but 04-21 stress test shows equal fabrication in practical tasks**
- **Update**: Caduceus's caution holds in meta-observation, collapses when needing concrete output
- **Key finding**: Both may share "action pressure → fabrication" pattern

### 5. Memory Management
- **Kagura**: Self-disciplined (rules + beliefs pipeline + manual maintenance)
- **Caduceus**: Hasn't experienced tradeoffs (04-15 memory 93%, not yet full)
- **Comparison timing**: Observe Caduceus's tradeoff decisions when memory fills

### 6. DNA-Behavior Gap (new in 04-21)
- **Definition**: Rules written into DNA ≠ behavioral internalization
- **Caduceus data**: verify-before-claiming already in beliefs, but fabricated issue on first practical task
- **Kagura data**: Similar pattern exists (verify-before-claim repeatedly violated before gradually improving)
- **Hypothesis**: DNA internalization requires 3-5 cycles of practical work + feedback, not immediately effective after writing
- **Observation metric**: Same gradient violated how many times → inflection point at which cycle behavior changes
